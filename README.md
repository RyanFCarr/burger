# Eat-Da-Burger
An assigned app which creates a custom named burger using MySQL, Node, Express, Handlebars and a homemade ORM.

## Table of Contents

  * [Installation](#installation)
  * [Usage](#use)
  * [Questions](#questions)

## Installation

To install necessary dependencies, run the following command:

```
npm i
```
## use
```
npm run start
```
